Decision Tree Builder API spec

Version 6

Type Aliases

<tree> := <node> | <leaf> <node> := { "type": "tree", "measurement": string, "threshold": number, "left": <tree>, "right": <tree> } <leaf> := { "type": "leaf", "diagnosis": string? } <marked_tree> := <node> | <marked_leaf> <marked_leaf> := <leaf> & { "marked": boolean }

Endpoints

  • GET /measurements
  • GET /athletes
  • GET /tree
  • POST /tree
  • GET /trees
  • POST /suggestion

    • GET /measurements

    returns string[]

    GET /athletes

    query { "limit": number?, "offset": number? }
    returns { "data": { "name": string, "measurements": { "descriptor": string, "value": number }[] }[], "next": string? }[]

    GET /tree

    query { "id": string }
    returns { "id": string, "name": string, "description": string, "tree": <tree> }

    POST /tree

    body { "name": string, "description": string, "tree": <tree> "log": string }

    GET /trees

    query { "limit": number?, "offset": number? }
    returns { "data": { "id": string, "name": string, "description": string, "amount_nodes": number }[], "next": string? }

    POST /suggestion

    body <marked_tree>
    returns <tree>
    The returned subtree is suggested to replace the marked leaf in the submitted marked tree. This method does not return whole trees.
